1 Configuration file format
2 =========================
3 
4 Atomic configurations are stored on disk by NNP applications in a simple ASCII
5 file. Data sets with training structures need to be provided in the same format.
6 The file name for input configurations is usually `input.data`. A configuration
7 file may contain multiple structures, each enclosed by the `begin` and `end`
8 keywords. The lines in between must begin with one of the following keywords:
9 
10 - `atom`
11 - `lattice`
12 - `comment`
13 - `energy`
14 - `charge`
15 
16 Here is a sample layout:
17 
18 ```
19 begin
20 comment <comment>
21 lattice <ax> <ay> <az>
22 lattice <bx> <by> <bz>
23 lattice <cx> <cy> <cz>
24 atom <x1> <y1> <z1> <e1> <c1> <n1> <fx1> <fy1> <fz1>
25 atom <x2> <y2> <z2> <e2> <c2> <n2> <fx2> <fy2> <fz2>
26 ...
27 atom <xn> <yn> <zn> <en> <cn> <nn> <fxn> <fyn> <fzn>
28 energy <energy>
29 charge <charge>
30 end
31 begin
32 ...
33 end
34 ...
35 begin
36 ...
37 end
38 ```
39 where the arguments of the keywords are:
40 
41 - `<comment>`: comment line
42 - `<ax>`...`<cz>`: box vectors @f$\vec{\mathbf{a}}, \vec{\mathbf{b}},
43  \vec{\mathbf{c}}@f$ (see nnp::Structure::calculateInverseBox()).
44 - `<x1>`...`<zn>`: atom coordinates of `n` atoms
45 - `<e1>`...`<en>`: atom element string (e.g. Cd, S)
46 - `<c1>`...`<cn>`: atom charge (for long range neural network only)
47 - `<n1>`...`<nn>`: not used
48 - `<fx1>`...`<fzn>`: force components of `n` atoms
49 - `<energy>`: total potential energy
50 - `<charge>`: total charge (for long range neural network only)
51 
52 The `lattice` section must be omitted for non-periodic structures. It is
53 possible to mix periodic and non-periodic structures. Also, configurations may
54 contain different numbers of atoms. If atoms in a periodic structure are
55 initially outside of the simulation box they will be automatically mapped back
56 into the box (see nnp::Structure::remap()). Here is an example configuration
57 file with 3 structures, 2 periodic and 1 non-periodic:
58 
59 ```
60 begin
61 comment This periodic structure contains 2 Cd and 2 S atoms.
62 lattice 1.0 0.0 0.0
63 lattice 0.0 1.0 0.0
64 lattice 0.0 0.0 1.0
65 atom 0.1 0.2 0.3 Cd -0.1 0.0 -0.1 -0.3 0.1
66 atom 0.2 0.4 0.8 Cd -0.1 0.0 -0.2 0.6 -0.6
67 atom 0.7 0.2 0.7 S 0.1 0.0 -0.8 -0.1 0.1
68 atom 0.1 0.1 0.4 S 0.1 0.0 1.1 -0.2 0.4
69 energy 123.456
70 charge 0.0
71 end
72 begin
73 comment This non-periodic structure contains 1 Cd and 2 S atoms.
74 atom 0.9 0.1 0.8 Cd -0.1 0.0 -0.3 -0.3 0.1
75 atom 0.7 0.2 0.2 S 0.1 0.0 -0.8 0.1 0.3
76 atom 0.6 0.9 0.4 S 0.1 0.0 1.1 0.2 -0.4
77 energy 1337.00
78 charge 0.0
79 end
80 begin
81 comment This periodic structure contains 3 Cd and 3 S atoms.
82 lattice 2.0 0.0 0.0
83 lattice 1.0 2.0 0.0
84 lattice 1.0 1.0 2.0
85 atom 1.9 0.2 1.7 S 0.1 0.0 0.4 -0.1 -0.2
86 atom 1.1 0.2 0.5 Cd -0.1 0.0 -0.1 -0.3 0.2
87 atom 0.2 1.4 0.8 Cd -0.1 0.0 -0.2 0.8 0.5
88 atom 0.9 0.2 1.7 S 0.1 0.0 -0.7 -0.3 -0.6
89 atom 0.8 1.2 0.1 Cd -0.1 0.0 -0.2 0.1 0.5
90 atom 0.1 0.1 0.4 S 0.1 0.0 0.8 -0.2 -0.4
91 energy 543.210
92 charge 0.0
93 end
