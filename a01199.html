Note
The implementation of both angular symmetry functions described above also supports an additional shift parameter in the exponential function (analogous to the radial symmetry function or the weighted symmetry functions below). Specifying the shift parameter is optional, omitting it will reproduce the formulae given above.

Weighted atom-centered symmetry functions

In 2018 an approach to overcome limitiations for systems with many different atom species was put forward here:

M. Gastegger, L. Schwiedrzik, M. Bittermann, F. Berzsenyi and P. Marquetand, J. Chem. Phys. 148, 241709 (2018) [https://doi.org/10.1063/1.5019667]
